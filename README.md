# Elva iOS SDK版本更新
## 1.1.6 (最新)
1、支持按照设备id提交客诉<br />
## 1.1.5
1、FAQ点赞，增加俄语、韩语、西班牙语、日语、阿拉伯语、泰语支持<br />
2、客服可以回复图片<br />
3、玩家消息是否已读标记

1、增加接口`setSDKLanguage`<br />
2、支持横竖屏显示时候可以强制竖屏显示

3、服务器serverId支持字符串<br />
4、评价框显示优化<br />
